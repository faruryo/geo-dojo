# UI Contract: 難易度バッジ（問題カード内）

本機能は外部 API を公開しない。ここでは `QuizRunner` 内に追加する難易度表示の UI 契約を定義する。

## 表示要素

- **要素**: shadcn/ui `Badge`（既存 import）1 個。
- **テキスト**: `DIFFICULTY_LABEL[difficulty]`（例: `☆☆ 中級`）。独自文字列を生成しない。
- **配置**: 問題カード（`rounded-xl bg-card ...` ブロック）内、問題文（`市区町村名` / 都道府県名）の近傍。進捗ヘッダー行（中断・`N / M`・`N 正解`）および残り秒数バーとは別ブロックで、重ならない。
- **対象モード**: A / B / C / D、および同じ `QuizRunner` を使う復習クイズ。

## 入力 → 表示の対応

| 問題 | 難易度の導出元 | 表示 |
|------|----------------|------|
| モードA（`kind: 'A'`） | `representativeDifficulty(instances)` | 値があればバッジ表示、`undefined` なら非表示 |
| モードB/C/D（`kind: 'BCD'`） | `municipality.difficulty`（= `representativeDifficulty([municipality])`） | 値があればバッジ表示、`undefined` なら非表示 |

## 振る舞い契約

- **C-01**: 問題が表示されている間、対応する難易度バッジが表示される（難易度データがある場合）。
- **C-02**: 次の問題へ進むと、バッジは新しい問題の難易度に更新される。
- **C-03**: 解答後のフィードバック表示中（`feedback !== 'idle'`）もバッジは表示され続け、消えたり別の値に変わったりしない。
- **C-04**: 難易度が導出できない（`undefined`）場合、バッジ領域は描画されず、カードのレイアウトが崩れない。
- **C-05**: バッジは正解（都道府県名・市区町村名）そのものを露出しない。表示するのは難易度区分のみ（FR-006）。
- **C-06**: 375px 幅・ダークモードで問題文・選択肢・地図の視認性を損なわない（SC-004）。

## 非対象

- 難易度の値の算出・変更（表示のみ）。
- 結果画面・設定画面の表示変更（本機能の対象外）。
